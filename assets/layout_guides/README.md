# Layout Guide Assets

Figmaボード: `figma://thumbnail-presets-v1`（社内共有リンク）

## 書き出しルール
1. 各プリセットごとに 1280x720 のガイド画像をPNGで出力し、本ディレクトリに `preset_<name>.png` として保存。
2. 余白や文字グリッドが分かるよう、背景は半透明レイヤーつきで書き出す。
3. 画像更新時は `docs/layout_guides/thumbnail_presets.md` の数値も合わせて更新。

> 現状はテキスト仕様のみ。Figmaガイドが完成次第 `preset_left_text.png` などを追加してください。
