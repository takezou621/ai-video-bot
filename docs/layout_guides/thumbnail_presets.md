# Thumbnail Layout Specs

テンプレごとの配置ルールとカラーパレット。`thumbnail_generator.py` の `LAYOUT_PRESETS` と `THEME_LIBRARY` は本ドキュメントをソースオブトゥルースとする。

## 1. L-Split (left_text_right_characters)
- **テキスト領域**: 左 90px から幅 52% を占有。行間 = フォントサイズ + 12px。
- **キャラ**: 右下に男女キャラ。男性を手前、女性を10〜20px上に。
- **装飾**: 下部グラデ + 上部バー。カテゴリバッジは左上。
- **用途**: 対話・解説の汎用構図。

## 2. Badge Stack
- **テキスト領域**: 左 120px、幅 50%。ジャンプ率を 1.7 倍確保。
- **縦バッジ**: 左端に 90px×260px のスタック。上半分に「POINT」など固定文字。
- **キャラ**: 右下クラスター、吹き出し必須。
- **用途**: 「3つの秘訣」「5 STEP」など Tips 系。

## 3. Center Focus
- **テキスト領域**: 中央寄せ、幅 60%、縦位置は中央。背景は斜めパターン。
- **キャラ**: 右にやや小さめで配置。余白を多めに。
- **用途**: キービジュアルを強調したいとき。

## 4. Before/After Split
- **テキスト領域**: 左 160px、幅 55%。
- **装飾**: 画面中央に縦ライン。上下に BEFORE/AFTER ラベル（英字）。
- **キャラ**: 左側にキャラを寄せ、右側は背景またはアイコン。
- **用途**: VS 比較、劇的ビフォーアフター。

## 5. Minimal (No Characters)
- **テキスト領域**: センター、幅 70%。行間はフォント + 18px。
- **キャラ**: なし。代わりにアイコン or 数字バッジ。
- **用途**: グラフや図解だけで訴求したいとき。

## QAチェック
1. メイン文字 = 8〜12文字、2行以内。
2. サブ文字 = 12〜20文字、最大2行。
3. 色数 = 4色 + 白黒以内（`thumb_lint.py` で検証）。
4. 150px縮小で文字が読める（edge score > 35）。
5. ファイルサイズ 80KB〜2MB。

> 上記に合わないサムネイルは `thumb_lint.py` で FAIL となるよう閾値を調整済み。
